# ALIGNMENT_UI_Optimization

## 1. 原始需求
- 优化圆形功能首页，提升信息层次和视觉质感。
- **信息层次**：主功能（圆心/大尺寸/明亮） vs 辅助功能（稍小/半透明）。
- **配色风格**：深蓝渐变星空背景，主按钮品牌色渐变+阴影，辅助按钮低饱和。
- **图标统一**：线性/双色风格，尺寸统一。
- **层级感**：按钮增加悬浮感（阴影、光晕、半透明）。
- **个性化信息**：加大欢迎语字体，校历提醒卡片化。
- **交互细节**：点击波纹、缩放动画。

## 2. 项目现状
- **布局**：ConstraintLayout 5个按钮呈五边形环绕中心 Logo。
- **风格**：已具备 Glassmorphism 和 粒子背景。
- **按钮**：统一尺寸 (90dp)，使用单一颜色图标，背景均为 `bg_glass_button`。
- **Header**：已存在，包含欢迎语和跑马灯，但样式较简单。
- **中心**：目前是一个显示校徽 Logo 的 ImageView。

## 3. 需求理解与决策
- **布局调整**：
  - 将 **“生活”** 选定为主功能，移至 **圆心位置**，尺寸放大 (120dp)。
  - 其余 4 个功能（学习、导航、通知、我的）环绕四周，尺寸保持或微调 (80-90dp)。
  - 原有的中心 Logo 需移除或融合进主功能按钮（此处将“生活”作为核心，Logo 可作为背景水印或顶部品牌标识）。鉴于“生活”是核心，我们将把原有的中心 Logo 区域改造为“生活”主按钮。
- **视觉升级**：
  - 创建新的 `bg_main_button.xml` (渐变+阴影) 给主按钮。
  - 创建新的 `bg_sub_button.xml` (半透明+描边) 给辅助按钮。
  - 优化 Header 区域，将跑马灯改为卡片样式。
- **交互增强**：
  - 保持现有的入场动画。
  - 增强点击反馈 (Scale + Ripple)。

## 4. 最终共识
- **中心**：放置“生活”大按钮 (主功能)。
- **四周**：放置 学习、导航、通知、我的 (辅助功能)。
- **样式**：差异化设计主辅按钮。
- **顶部**：优化欢迎语排版和通知卡片。
